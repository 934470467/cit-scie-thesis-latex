# Overleaf 新手开箱即用教程

对于 95% 的同学，我们强烈推荐使用 **Overleaf** 免费版！

## 操作步骤
1. **获取源码**：将本项目的代码打包下载为 ZIP 格式。
2. **登录导入**：访问 [Overleaf 官网](https://www.overleaf.com/)，注册并登录。点击 `New Project` -> `Upload Project`，将 ZIP 包上传。
3. **编译器设置**：进入项目后，点击左上角的 `Menu`，将 **Compiler** 修改为 `XeLaTeX`。（必须设置为 XeLaTeX，否则无法正常显示中文）。
4. **开始编写**：在左侧文件树中找到 `report.tex` 并打开，你就可以开始愉快地写论文了！
5. **编译与预览**：点击右侧的 `Recompile` 按钮（或按 `Ctrl + Enter` / `Cmd + Enter`）即可生成 PDF。

## ⚠️ 注意事项
- **大文件超时**：Overleaf 免费版对编译时间有限制。如果你的插图过多或分辨率极大，可能会引发编译超时报错。建议压缩图片后再上传。
- **网络合规**：请确保你的参考文献链接能够在国内正常访问。
